## Setup
1. Install JSON Server: `npm install -g json-server`
2. Start backend: `npx json-server --watch data/db.json --port 3000`
3. Open `index.html` in browser

## Features
- Browse and filter recipes by dietary category
- Add new recipes via form with inline validation
- Weekly meal planner with localStorage persistence
- Grocery list with categories, add/delete/check off items
- Admin panel: view all recipes, edit, delete, hide from users
- Live stats: total recipes, hidden count, avg calories, avg rating
